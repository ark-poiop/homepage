# 포트폴리오 개선 제안

현재 완성된 구조를 기반으로 추가하면 임팩트가 큰 요소들을 기능/UX 두 축으로 정리했습니다.

---

## 1. 인터랙션 & 애니메이션

### 스크롤 진입 애니메이션 (Scroll Reveal)
카드나 섹션이 뷰포트에 들어올 때 페이드인/슬라이드업 효과.
`IntersectionObserver` API로 구현 가능 (라이브러리 불필요).
→ 페이지가 살아있다는 느낌을 주는 가장 효과적인 방법.

### Hero 타이핑 효과
"일상의 불편함을 AI로 해결합니다" 문구가 한 글자씩 타이핑되는 효과.
→ 첫인상에서 시선을 잡아두는 역할.

### 프로젝트 카드 Hover 미세 효과 강화
현재 `translateY(-2px)` 외에 카드 내 이모지 살짝 바운스 효과 추가.
→ 클릭 가능하다는 어포던스를 직관적으로 전달.

---

## 2. 네비게이션 & 탐색성

### 활성 섹션 하이라이트 (Active Nav)
스크롤 위치에 따라 현재 보고 있는 섹션의 네비 링크를 강조 표시.
`IntersectionObserver`로 각 섹션 진입 감지.
→ 사용자가 현재 위치를 인지할 수 있어 탐색 경험 향상.

### 맨 위로 버튼 (Back to Top)
스크롤이 일정 이상 내려가면 우하단에 부드럽게 나타나는 버튼.
→ 긴 페이지에서 필수적인 편의 요소.

### 프로그레스 바
페이지 상단에 읽기 진행도를 나타내는 얇은 바(1~2px).
→ 미니멀 디자인과 잘 어울리는 세련된 디테일.

---

## 3. 프로젝트 섹션 강화

### 카테고리 필터 탭
"전체 / 자동화 / 생산성 / 생활 편의" 등 태그 기반 필터 버튼.
클릭 시 해당 카테고리 카드만 표시 (페이드 전환).
→ 프로젝트가 많아질수록 탐색 편의성이 크게 올라감.

### 프로젝트 상태 배지
모달 또는 카드에 `완성` / `진행 중` / `아이디어` 배지 추가.
→ 작업 흐름이 보여져 기획자로서의 신뢰감 형성.

### 모달 내 데모 링크 / 스크린샷
모달 하단에 "데모 보기" 버튼 또는 간단한 이미지 미리보기 영역 추가.
→ 글보다 시각적 증거가 설득력 높음.

---

## 4. About / 신뢰도 요소

### 타임라인 섹션
프로젝트를 연도·월 기준으로 나열하는 타임라인 뷰.
→ 꾸준히 만들어온 사람임을 시각적으로 증명.

### 추천사 / 한 줄 후기 (Testimonials)
실제 사용자나 협업자의 짧은 후기 슬라이더 (2~3개).
→ 포트폴리오의 신뢰도를 높이는 가장 강력한 요소 중 하나.

### 관심사 태그 클라우드
AI, 자동화, 생산성 도구, 일상 해킹 등 키워드를 시각화.
→ 기획 방향성과 개성을 한눈에 전달.

---

## 5. 접근성 & 사용성

### 다크 모드 토글
헤더 우측에 🌙 / ☀️ 버튼으로 다크/라이트 테마 전환.
CSS 변수 기반이라 색상 팔레트만 교체하면 구현 가능.
→ 현재 CSS 변수 구조가 이미 이 작업에 최적화되어 있음.

### OG 메타 태그 (SNS 공유 최적화)
`og:title`, `og:description`, `og:image` 추가.
→ 링크를 공유할 때 카드형 미리보기가 보이도록.

### 한국어 / 영어 언어 토글
콘텐츠를 두 언어로 준비해 우측 상단 버튼으로 전환.
→ 글로벌 기회를 열어두는 데 효과적.

---

## 6. 연락처 & 전환율

### 간단한 문의 폼
이름 + 메시지 입력 후 EmailJS 또는 Formspree로 이메일 발송.
→ 이메일 클라이언트를 열지 않아도 되어 전환율 향상.

### 복사 버튼 (이메일)
이메일 주소 옆 📋 아이콘 클릭 시 클립보드에 복사 + "복사됨!" 토스트 알림.
→ 모바일에서 특히 유용한 UX.
